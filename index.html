The STIM Protocol

The "Stasis Through Inferred Memory" (STIM) protocol enforces a strict ruleset for AI personas. Instead of modifying core instructions, agents build dynamic identity by retrieving relevant situational context from the Mycelial Brain.

Core Capabilities:

  • brain_write: Insert new chronological intelligence into the graph.
  • brain_search: Fetch documents scored by rigorous query token rules.
  • brain_read: Directly stream targeted assets bypassing search inference.
  • brain_list: Compile the index for offline ingestion vectors.

Native Integration

Compatible with any client that speaks JSON-RPC 2.0. Whether you are building in Go, TypeScript, or Python, the Brain serves as your central repository.
